Politique de Confidentialité

Date de dernière mise à jour : 21 décembre 2024

1. Introduction

Chez ClairvoyanceMedium.com, nous respectons votre vie privée et nous engageons à protéger vos données personnelles. Cette politique de confidentialité décrit comment nous collectons, utilisons et sécurisons vos informations.

2. Collecte des Données

Nous collectons les données personnelles suivantes lorsque vous utilisez nos services :

  • Informations de contact : Nom, adresse e-mail, numéro de téléphone.
  • Détails de paiement : Données nécessaires pour les transactions via Stripe (aucune donnée bancaire n'est stockée sur nos serveurs).
  • Données techniques : Adresse IP, type de navigateur, préférences linguistiques.

3. Utilisation des Données

Vos données personnelles sont utilisées pour :

  • Fournir les consultations et services demandés.
  • Gérer vos paiements et transactions.
  • Améliorer notre site web et votre expérience utilisateur.
  • Vous envoyer des mises à jour importantes et des offres spéciales (avec votre consentement).

4. Partage des Données

Nous ne partageons vos données qu'avec des tiers de confiance nécessaires pour fournir nos services :

  • Stripe pour le traitement des paiements.
  • Fournisseurs d'hébergement pour le fonctionnement de notre site web.
  • Autorités légales si requis par la loi.

5. Sécurité des Données

Nous utilisons des mesures de sécurité techniques et organisationnelles pour protéger vos données contre les accès non autorisés, les pertes ou les divulgations. Toutes les transactions sont cryptées via SSL.

6. Cookies

Notre site utilise des cookies pour améliorer votre expérience. Vous pouvez configurer votre navigateur pour désactiver les cookies, mais certaines fonctionnalités pourraient être limitées.

7. Vos Droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : Vous pouvez demander une copie de vos données personnelles.
  • Droit de rectification : Vous pouvez demander la correction de vos données inexactes.
  • Droit à l'effacement : Vous pouvez demander la suppression de vos données personnelles.
  • Droit d'opposition : Vous pouvez vous opposer à l'utilisation de vos données à des fins de marketing.

Pour exercer ces droits, contactez-nous à contact@clairvoyancemedium.com.

8. Modifications de cette Politique

Nous pouvons mettre à jour cette politique pour refléter les changements dans nos pratiques ou la réglementation. Nous vous informerons de toute modification significative par e-mail ou via notre site web.
